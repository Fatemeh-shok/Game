## 🧠 About The Project

**The Skin Maker II** is a game landing page concept designed with a dark horror atmosphere.
This project focuses on visual storytelling, layout design, and landing page structure.

The main goal of this project is to practice:
- UI layout design
- Dark theme styling
- Landing page composition for games

---

## ✨ Features

- 🎭 Horror-style dark UI
- 🧭 Navigation menu
- 🖤 Hero section with game intro
- ⭐ Add to favorite section
- ▶️ Play & Watch Trailer buttons
- 🧱 Structured layout with sections
- 📱 Responsive design (in progress)

---

  - Flexbox
